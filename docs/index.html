Menu from HTML: Click Interaction

Open menu items with a click. This is the default mode.

import { menuFromHTML } from "@evermade/menu-toolkit"; const menuEl = document.querySelector('.js-main-menu'); if (menuEl) { const menu = menuFromHTML( menuEl, { action: 'click' } ); }

This is demo for the @evermade/menu-toolkit.
